## Getting Started

```bash
$ npm install
```

## Running the app

```bash

# watch mode
$ npm run start:dev

# install postgres
$ docker compose up dev-db -d
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
