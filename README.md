## Installation
Install the npm dependencies
```bash
$ npm install
```

## Running the app

```bash
$ npm run start:dev
```

## Start PostgreSQL docker container
```
npm run db:dev:up
```

## Start Prisma Studio
```
npx prisma studio
```
